Add a falling snow effect to your WordPress site.

== Description ==

JAM jSnow adds a falling snow effect to your WordPress site. It uses the jSnow jQuery plugin to create the effect.

== Installation ==

Upload the JAM CSS Naked Day plugin to your blog, Activate it, then set up your options (or leave the defaults).
